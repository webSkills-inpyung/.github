# 인평자동차고등학교 AI소프트웨어과 전공심화동아리

기능경기대회 풀이는 **모듈마다 레포를 각자 생성**해서, 아래 네이밍 규칙에 맞게 보관해주세요.
(한 사람이 A·B·C 모듈을 풀면 → 레포 3개)

## 레포 네이밍 규칙
```
{연도}-{대회}-{종목}-{이니셜}-{모듈}
```

| 자리 | 값 | 설명 |
|---|---|---|
| 연도 | `2026` | 연도 |
| 대회 | `nat` / `reg` | 전국 / 지방(인천) |
| 종목 | `web` / `app` | 종목 |
| 이니셜 | `ytw`, `jmj` … | 이름 약자 (소문자) |
| 모듈 | `a` / `b` / `c` | 모듈 |

**예시 — 윤태웅, 전국 web:**
```
2026-nat-web-ytw-a
2026-nat-web-ytw-b
2026-nat-web-ytw-c
```

> 이니셜을 모듈보다 앞에 두어, 한 사람의 A·B·C 레포가 목록에서 나란히 정렬됩니다.
