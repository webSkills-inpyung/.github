# 인평자동차고등학교 AI소프트웨어과 전공심화동아리

기능경기대회 풀이는 **개인별로 레포 하나**를 만들어, 그 안에서 문제·모듈을 폴더로 정리해주세요.
레포를 여러 개 만들 필요 없이, **본인 레포 1개에 다 담습니다.** (진입장벽 최소화)

## 레포 네이밍 규칙 (개인별 1개)
```
{연도}-{대회}-{종목}-{이니셜}
예) 2026-nat-web-ytw
```

| 자리 | 값 | 설명 |
|---|---|---|
| 연도 | `2026` | 연도 |
| 대회 | `nat` / `reg` | 전국 / 지방(인천) |
| 종목 | `web` / `app` | 종목 |
| 이니셜 | `ytw`, `jmj` … | 이름 약자 (소문자) |

## 레포 안 폴더 구조 (파일 정리)
```
2026-nat-web-ytw/
├── README.md            # 사용 언어·프레임워크·환경·실행법
├── 1번문제/
│   ├── A-Module/
│   ├── B-Module/
│   └── C-Module/
├── 2번문제/
│   ├── A-Module/  B-Module/  C-Module/
└── 3번문제/
    ├── A-Module/  B-Module/  C-Module/
```
> 문제 3개 × 모듈 A·B·C는 **레포가 아니라 폴더**로 나눕니다.

## readme 필수
레포 최상단에 **README.md** 를 만들어 사용한 언어·프레임워크·환경·실행법을 기재해주세요.

> ex) html, css, javascript, xampp(version x.x.x) / 실행: xampp 켜고 htdocs에 복사 후 localhost 접속
