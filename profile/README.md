# 인평자동차고등학교 AI소프트웨어과 전공심화동아리

기능경기대회 풀이는 **문제·모듈마다 레포를 각자 생성**해서, 아래 네이밍 규칙에 맞게 보관해주세요.
(한 사람이 문제 3개 × 모듈 A·B·C = **레포 9개**)

## 레포 네이밍 규칙
```
{연도}-{대회}-{종목}-{이니셜}-{문제}{모듈}
```

| 자리 | 값 | 설명 |
|---|---|---|
| 연도 | `2026` | 연도 |
| 대회 | `nat` / `reg` | 전국 / 지방(인천) |
| 종목 | `web` / `app` | 종목 |
| 이니셜 | `ytw`, `jmj` … | 이름 약자 (소문자) |
| 문제 | `1` / `2` / `3` | 대회문제 번호 |
| 모듈 | `a` / `b` / `c` | 모듈 |

**예시 — 윤태웅, 전국 web:**
```
2026-nat-web-ytw-1a   2026-nat-web-ytw-1b   2026-nat-web-ytw-1c
2026-nat-web-ytw-2a   2026-nat-web-ytw-2b   2026-nat-web-ytw-2c
2026-nat-web-ytw-3a   2026-nat-web-ytw-3b   2026-nat-web-ytw-3c
```
> `1a` = 1번 문제 A모듈. 이니셜·문제를 모듈보다 앞에 두어, 한 사람의 레포가 문제·모듈 순으로 나란히 정렬됩니다.

## readme 필수
각 풀이 레포에는 **readme.md** 를 만들어 사용한 언어·프레임워크·환경을 자세히 기재해주세요.

> ex) html, css, javascript, xampp 사용 (version x.x.x)
